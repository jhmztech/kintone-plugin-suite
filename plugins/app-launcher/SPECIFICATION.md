# App Launcher Plugin

## Version

0.1.0

## Purpose

Provide configurable buttons that launch other Kintone apps.

---

## Features

- Configurable button labels
- Configurable App IDs
- Optional View ID
- Optional Custom URL
- Open in new tab

---

## Future Features

- Icons
- Button colors
- Security groups
- Open specific records
- Pass search parameters
- Space Field support
- Mobile support

---

## Notes

This plugin will not modify any Kintone records.

Its purpose is navigation only.
